# Accessibility Store

## Overview
The Accessibility Store is a web application designed to provide an inclusive shopping experience for all users, including those with disabilities. The application features a clean and accessible design, ensuring that all users can navigate and interact with the site effectively.

## Project Structure
- **index.html**: The main entry point of the website, linking to other pages.
- **products.html**: Displays a list of products available in the store, utilizing AJAX to fetch product data.
- **cart.html**: Shows items added to the shopping cart, allowing users to view and manage their selected products.
- **css/style.css**: Contains styles for the website, ensuring accessibility and visual appeal.
- **js/jquery.min.js**: The jQuery library for DOM manipulation, event handling, and AJAX requests.
- **js/main.js**: Custom JavaScript code for handling user interactions, AJAX calls, and dynamic content loading.
- **data/products.json**: Contains product data in JSON format for populating product listings.

## Setup Instructions
1. Clone the repository to your local machine.
2. Open the `index.html` file in your web browser to view the application.
3. Ensure that you have a local server running if you are using AJAX to fetch data from `products.json`.

## Features
- Accessible navigation and layout for users with disabilities.
- AJAX functionality to dynamically load product data.
- Shopping cart management for a seamless shopping experience.
- Responsive design to ensure usability across various devices.
